# Ejemplo de POO: Ventilador Inteligente

Este proyecto es un ejemplo de **POO en Java** usando **clases abstractas** e **interfaces**.  
Simula un **ventilador inteligente**, que puede encenderse, apagarse, conectarse a Wi-Fi y ajustar la velocidad.

## Características
- Uso de clases abstractas (`DispositivoInteligente`)
- Implementación de interfaces (`Conectividad`)
- Métodos que representan funciones reales del dispositivo
- Ejecución simple desde la clase `Main`

## Estructura del proyecto
```
src/
 └── smarthome/
     ├── Conectividad.java
     ├── DispositivoInteligente.java
     ├── VentiladorInteligente.java
     └── Main.java
```

## Ejecución
1. Abre el proyecto en IntelliJ IDEA o VS Code.
2. Compila y ejecuta la clase `Main.java`.
3. Observa las acciones del ventilador inteligente en consola.
