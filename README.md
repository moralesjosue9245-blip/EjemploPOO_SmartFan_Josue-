# EjemploPOO_SmartFan_Josue-
Descripción del proyecto:
Este proyecto demuestra los principios de la Programación Orientada a Objetos (POO) en Java, utilizando clases abstractas e interfaces.
El programa simula un ventilador inteligente, capaz de encenderse, apagarse, conectarse a Wi-Fi y ajustar su velocidad o modo de funcionamiento.
