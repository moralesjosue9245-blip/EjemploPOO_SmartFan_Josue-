# Sales Project (Angular)

Proyecto Angular de ejemplo entregable — generado automáticamente.

## Requisitos
- Node.js (>= 16 recommended)
- npm (comes with Node)
- Angular CLI (opcional). Si no la tienes instalada globalmente, usa `npx`.

## Pasos para ejecutar
1. Descomprime el zip en una carpeta.
2. Abre una terminal en la carpeta del proyecto.
3. Ejecuta:
   ```bash
   npm install
   ```
4. Para iniciar la aplicación:
   ```bash
   npx ng serve --open
   ```
   o si tienes Angular CLI global:
   ```bash
   ng serve --open
   ```
5. La aplicación abrirá en: http://localhost:4200

## Nota
- El repositorio contiene un ejemplo básico con un componente `sales-person-list` que muestra una tabla.
- Si necesitas que lo empaque con dependencias (node_modules), dime y te explico por qué normalmente no se incluye en un zip.
